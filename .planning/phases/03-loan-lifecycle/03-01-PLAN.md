현재 Loan 모델 (관련 부분):
```prisma
model Loan {
  id              String         @id @default(cuid())
  organizationId  String
  ...
  loanAmount      Decimal        @db.Decimal(15, 0)
  balance         Decimal        @db.Decimal(15, 0)
  interestRate    Decimal        @db.Decimal(5, 2)
  repaymentType   String
  loanTermMonths  Int
  startDate       DateTime       @db.Date
  endDate         DateTime       @db.Date
  status          String         @default("ACTIVE")
  overdueStage    String         @default("NORMAL")
  overdueDays     Int            @default(0)
  memo            String?
  createdAt       DateTime       @default(now())
  updatedAt       DateTime       @updatedAt
  schedules       LoanSchedule[]
  payments        Payment[]
  @@unique([organizationId, loanNumber])
  @@map("loans")
}
```

Setting 모델은 변경하지 않는다. org 수준 기본 수수료율은:
- key: "prepayment_fee_rate"
- value: "0" (기본값)
- label: "중도상환수수료율 (%)"
으로 Setting 레코드를 삽입/조회하는 방식을 사용한다.
