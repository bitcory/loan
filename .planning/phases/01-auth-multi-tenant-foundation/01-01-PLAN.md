schema.prisma를 다음과 같이 수정한다:

1. generator client 블록 위에 다음 두 모델을 추가:

```prisma
model Organization {
  id           String    @id @default(cuid())
  name         String
  slug         String    @unique
  createdAt    DateTime  @default(now())
  updatedAt    DateTime  @updatedAt
  users        User[]
  customers    Customer[]
  collaterals  Collateral[]
  mortgages    Mortgage[]
  loans        Loan[]
  loanSchedules LoanSchedule[]
  payments     Payment[]
  settings     Setting[]

  @@map("organizations")
}

model User {
  id             String       @id @default(cuid())
  organizationId String
  organization   Organization @relation(fields: [organizationId], references: [id])
  username       String       @unique
  passwordHash   String
  name           String
  role           String       @default("STAFF") // ADMIN, STAFF
  isActive       Boolean      @default(true)
  createdAt      DateTime     @default(now())
  updatedAt      DateTime     @updatedAt

  @@map("users")
}
```

2. Customer 모델 수정:
   - `customerNumber String @unique` → `customerNumber String` (전역 unique 제거)
   - 필드 목록 상단에 추가: `organizationId String?` / `organization Organization? @relation(fields: [organizationId], references: [id])`
   - 모델 하단(@@map 위)에 추가: `@@unique([organizationId, customerNumber])`

3. Collateral 모델 수정:
   - 필드 목록 상단(customerId 위)에 추가: `organizationId String?` / `organization Organization? @relation(fields: [organizationId], references: [id])`

4. Mortgage 모델 수정:

5. Loan 모델 수정:
   - `loanNumber String @unique` → `loanNumber String` (전역 unique 제거)
   - 필드 목록 상단에 추가: `organizationId String?` / `organization Organization? @relation(fields: [organizationId], references: [id])`
   - 모델 하단(@@map 위)에 추가: `@@unique([organizationId, loanNumber])`

6. LoanSchedule 모델 수정:
   - 필드 목록 상단(loanId 위)에 추가: `organizationId String?` / `organization Organization? @relation(fields: [organizationId], references: [id])`

7. Payment 모델 수정:

```typescript
import "next-auth";
import "next-auth/jwt";

declare module "next-auth" {
  interface Session {
    user: {
      userId: string;
      organizationId: string;
      role: "ADMIN" | "STAFF";
      name?: string | null;
    };
  }
  interface User {
    organizationId: string;
    role: "ADMIN" | "STAFF";
  }
}

declare module "next-auth/jwt" {
  interface JWT {
    userId: string;
    organizationId: string;
    role: "ADMIN" | "STAFF";
  }
}
```
